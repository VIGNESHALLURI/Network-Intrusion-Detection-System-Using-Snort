# Network Intrusion Detection System Using Snort

This project is a basic Network Intrusion Detection System (NIDS) built using Snort on Ubuntu in VirtualBox.

## Features
- Installed and configured Snort
- Created custom ICMP detection rules
- Detected ping packets in real time
- Monitored network traffic

## Tools Used
- Ubuntu
- Snort
- VirtualBox
- Linux Commands

## Commands Used

Check Snort version:
```bash
snort -V
